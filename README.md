# Unique Driftwood Creations

A premium, highly aesthetic, modern responsive website for Boniface Chikwenhere's wood sculpting business, **Unique Driftwood Creations**, showcasing the rich, earth-toned culture of Southern Africa.

## Project Structure
- `index.html`: Contains semantic page layouts, structured sections, and detailed SVGs.
- `styles.css`: Dark-mode art-gallery styling with smooth typography, CSS grids, custom properties, and animations.
- `script.js`: Handles interactive transitions, dynamic sculpture category filtering, lightbox view, and email catalog inquiry.

## Sourcing
The images used in the gallery are linked directly to Boniface's original website assets hosted on Wix.

## Customization
To add your own images:
1. Save your image files in an `assets/` folder.
2. Update the `sculptures` array in `script.js` with the relative file paths and titles of your pieces.
